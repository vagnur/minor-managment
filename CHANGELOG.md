# Changelog

Todos los cambios importantes de este proyecto serán documentados en este archivo.

El formato sigue una versión simplificada de Keep a Changelog
y el versionado es de tipo semántico.

---

## [v0.1] - 2026-03-05

### Agregado

- Estructura base del proyecto
- Interfaz gráfica inicial con CustomTkinter
- Sistema de navegación entre módulos
- Implementación completa del módulo Postulación
- Generación automática de formularios Word desde Excel
- Creación automática de carpetas por carrera
- Validación básica de columnas del archivo Excel

### Infraestructura

- Inicialización del repositorio Git
- README inicial del proyecto
- Archivo requirements.txt
- Organización modular del código

## v0.2 - 2026-03-11

### Nuevo
- Implementación del módulo de Aceptación
- Generación automática de documento Word institucional de ingreso al Minor

### Mejoras
- Mejora en formato de tablas DOCX mediante clonación de filas modelo
- Heurística mejorada para separación de nombres completos
- Normalización de facultades
- Generación de texto institucional de semestre
- Validaciones adicionales de plantilla y Excel

### Infraestructura
- Registro centralizado de módulos
- Mejor organización modular del proyecto
